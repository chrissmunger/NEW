# 存储库



Cspace是一个免费和开源、加密和基于奖励的社交网络平台。我们的路线图、代码、项目管理系统等都驻留在GitHab中。​



**存储库​**

CSPACE 存储库包含多个 git 子模块存储库：​

引擎- 后端代码和 API​

前端- 客户端 Angular2 Web 应用程序​

Sockets - 用于实时通信的 WebSocket 服务器​

移动- React Native 移动应用​
